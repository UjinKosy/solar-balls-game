## Предусловия

- `pnpm install` выполнен.
- `docker compose up -d postgres` запущен (если фича трогает API).

## Запуск

```bash
pnpm dev
```

Открыть `http://localhost:5173`.

## Сценарий проверки

1. В главном меню нажать «Играть».
2. Перетащить карту «Солнце» на клетку (2, 3).
3. Убедиться: через 7 с появляется оранжевый шар.
4. Клик по шару — счётчик энергии увеличился на 25.
