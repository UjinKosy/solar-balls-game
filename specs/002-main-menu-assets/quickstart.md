# Quickstart: Главное меню и workflow ассетов

**ID фичи:** 002-main-menu-assets

> Минимальная инструкция, как локально увидеть фичу и проверить критерии приёмки.

---

## Предусловия

- Зависимости уже установлены.
- Проект запускается локально после этапа 0.

## Запуск

```bash
pnpm dev
```

Открыть URL фронтенда (по умолчанию `http://localhost:5173`).

## Сценарий проверки

1. Убедиться, что открылось главное меню с 5 кнопками:
   - ИГРАТЬ
   - КОРАБЛЬ АСТРОЧЕЛА
   - МАГАЗИН АСТРОЧЕЛА
   - НАСТРОЙКИ
   - ВЫХОД
2. Нажать `ИГРАТЬ` -> убедиться, что открылась сцена-заглушка и есть кнопка `Назад`.
3. Нажать `Назад` -> убедиться, что вернулись в главное меню.
4. Повторить шаги 2-3 для `КОРАБЛЬ АСТРОЧЕЛА`, `МАГАЗИН АСТРОЧЕЛА`, `НАСТРОЙКИ`.
5. Переключить язык через иконки флагов в углу:
   - выбрать RU -> убедиться, что все тексты меню/заглушек на русском;
   - выбрать EN -> убедиться, что все тексты меню/заглушек на английском.
6. Нажать `ВЫХОД` -> убедиться, что выполняется согласованное поведение:
   - пользователь выходит из текущей сессии;
   - выполняется переход на экран авторизации.
7. Проверить визуальные ассеты: фон меню, две разные неон-кнопки, персонажный элемент присутствуют и отображаются без артефактов.

## Ожидаемый результат

- Все пункты AC-1..AC-9 из `spec.md` выполняются.
- Навигация между меню и заглушками стабильна.
- Переключение ru/en работает на каждом экране этапа.
