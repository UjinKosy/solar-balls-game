# /constitution

Прочитай файл [`.specify/memory/constitution.md`](../../.specify/memory/constitution.md) целиком и выведи его пользователю. Это единственный источник правды для принципов проекта. Если пользователь после этого обсуждает фичи — держи принципы в голове и проверяй предлагаемые решения на соответствие каждому из 12 пунктов.

Если пользователь хочет изменить конституцию:
1. Обсуди, какой принцип затрагивается и почему.
2. Предложи точечное изменение в `.specify/memory/constitution.md`.
3. Напомни, что правка должна пройти как отдельный PR с меткой `governance` и пересмотром всех активных спек.
